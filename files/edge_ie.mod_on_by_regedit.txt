To enable the Internet Explorer (IE) mode button in Microsoft Edge via the registry, create a DWORD value InternetExplorerIntegrationReloadInIEModeAllowed set to 1 under HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Edge. This forces the option to appear in the menu, allowing you to reload sites, even if the setting is hidden in the GUI. 
Registry Method Steps:
	Open Registry Editor (regedit).
	Navigate to: HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Edge
	If the "Edge" key does not exist, right-click "Microsoft", select New > Key, and name it Edge.
	Right-click on Edge > New > DWORD (32-bit) Value.
	Name it: InternetExplorerIntegrationReloadInIEModeAllowed
	Double-click the new value and set Value data to 1.

Optional but recommended: 
	Create another DWORD named InternetExplorerIntegrationLevel and set it to 1.
	Restart Microsoft Edge. 

Alternative Registry Path:
	HKEY_CURRENT_USER\SOFTWARE\Policies\Microsoft\Edge can be used if the HKLM path does not work or if you want to apply it only for the current user. 
	Verifying the Button:
		Once configured, go to edge://settings/defaultbrowser:
			1. Ensure "Allow sites to be reloaded in Internet Explorer mode" is set to "Allow". 
			2. You can then add the button to your toolbar via settings or by clicking the three-dot menu > 
			3. Reload in Internet Explorer mode.
